## 에러 History

Github Pages로 페이지를 만드려고 했더니 에러가 계속 발생했다.

1.  next.config.js의 output:'export'를 추가해줘야한다.
2.  Next 14는 next config yml 파일의 `Static HTML ~~~~ next export` 옵션이 필요없다고 한다.
3.  나는 13을 사용중인데 git setting에서 yml 파일을 수정하려 해도 수정이 안된다는 에러를 계속 던졌다.

    - git 에서 계속 이슈가 생성되는 내용인데 실시간으로 계속해서 수정할 수 없다는 에러가 발생한다고 한다.(아직 명확한 방법을 찾지 못한거 같다.)

4.  그래서 직접 프로젝트에서 .github/workflow/yml 파일에서 위 내용을 지워서 push를 했더니 static 파일들을 정상적으로 불러오지 못해서 스타일이 다 꺠져버렸다.
